ToMiddayUtcDateTime(DateOnly)

Provides an equivalent System.DateTime to the supplied System.DateOnly with the time portion set to midday (12:00:00) UTC. This is used to minimise the possibility of dates being misinterpreted as the next or previous day due to the use of non-UTC timezones.
